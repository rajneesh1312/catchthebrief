## ONE-LINE BRIEF
CatchTheBrief (catchthebrief.com) is an automated "Content-to-Commerce" static site
that publishes AI-curated tech news summaries + daily deals every morning for Indian
buyers. Zero server. Zero monthly cost. Runs on GitHub Actions + GitHub Pages.

---

## TECH STACK
- Language:     Python 3.11
- AI:           Google Gemini 2.5 Flash (free tier, multiple API keys rotated)
- Frontend:     Pure HTML + CSS (no framework). Fonts: Playfair Display + DM Sans + DM Mono
- Hosting:      GitHub Pages (free)
- Automation:   GitHub Actions cron job (daily at 3:30 AM UTC = 9:00 AM IST)
- Dependencies: requests, beautifulsoup4, google-genai  (NO lxml — causes Windows build error)

## AD SLOTS — HOW TO ACTIVATE
All 3 slots are built into templates/index.html as placeholder divs.
To activate with real ads:
1. Sign up for Google AdSense or Media.net
2. Get your ad unit code
3. Replace the <div class="ad-placeholder">...</div> inside each ad slot
   with your actual ad script tag
4. Push to GitHub — ads go live automatically

Slot locations in index.html:
- Top banner: search "ad-banner-top"
- Mid-feed: search "ad-mid-feed"
- Sidebar: search "ad-sidebar"
